# Sistema de Leilão - LeiloesTDSat

## Descrição do Projeto
Este projeto visa a implementação de um sistema informatizado para gerenciar rotinas de uma casa de leilões. 
Através de uma interface simples e funcional efetua o registro/cadastro de "produtos" e utilza o MySQL Workbench para o armazemento de dados (persistência).

## Tecnologias Utilizadas
* Java
* MySQL
* Git/GitHub
